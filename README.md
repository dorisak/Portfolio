# Portfolio

Portfolio interactif présentant mes projets en Product Management IA & Santé.

## 🚀 Site en ligne

[Voir le portfolio](https://www.dorisailab.com)

## 🛠 Stack technique

- React 19 + TypeScript
- Vite
- Tailwind CSS
- Framer Motion

## 💻 Développement local

```bash
npm install
npm run dev
```

Le site sera accessible sur `http://localhost:3000`

## 📦 Build

```bash
npm run build
```
